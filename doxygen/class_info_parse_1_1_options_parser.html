Adds an option with T type parameter to be stored and later invoked to do its parsing.

Template Parameters
TType for the exported value
Parameters
nameThe LONG name of the option. SHORT name is deduced from the first character of the LONG name.
exporterA pointer to a memory block of type T, into which the parsed value will be put
Note
nullptr for exporter is not checked, yet
T must support operator>> from istream, this is made sure by SFINAE so it will die compile time
The creation of the SHORT name doesn't check if there already exists an option with the same SHORT name, thus it can be shadowed by earlier created options' SHORT names
