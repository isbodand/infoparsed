Adds an option with T type parameter to be stored and later invoked to do its parsing.

Template Parameters
TType for the exported value
Parameters
[in]nameThe LONG name of the option. SHORT name is deduced from the first character of the LONG name.
[out]exporterA pointer to a memory block of type T, into which the parsed value will be put
Note
nullptr for exporter is not checked, yet
T must support operator>> from istream, and operator<< from ostream; this is made sure by SFINAE so it will die compile time
The following is advised, otherwise the library doesn't define any explicit behaviour
T inVal, outVal;
std::stringstream ss; // or any stream, i && o stream is only
// required for this example
(ss << inVal) >> outVal;
inVal == outVal;
Option names are not checked, so one of the options will shadow the other, depending their position in the std::map
OptionAdder info::parse::OptionsParser::addOptions ( )
inline

Interface.

Adds multiple options to the parser.

Returns an instance of an internal OptionAdder object which offers an operator() which returns a reference to object allowing chain calling.

parser.addOptions()
("alpha|a", &a)
("beta|b", &b)
("silent|quiet|s|q", &stfu)
;
Returns
An object to call operator() on.
