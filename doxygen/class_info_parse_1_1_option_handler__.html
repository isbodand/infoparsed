Detailed Description

template<class T>
class InfoParse::OptionHandler_< T >

Groups multiple Option_<T>-s together for some type T Can execute handling of argument string on all contained Option_<T>-s.

Used internally, not really useful outside of the internals of the project

Template Parameters
TThe type for Option_<T>-s can be accepted into the group
