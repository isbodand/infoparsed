49  /**
50  * Stores the acceptable
51  * names of an option by which it can be
52  * parsed.
53  *
54  * For example both short and long variants
55  * of the same option are stored, however
56  * arbitrary names can be stored for one option,
57  * like all four for one option: `--quiet --silent -q -s`
58  *
59  * OptionString can be constructed implicitly from
60  * a std::string and from C-String literals(const char*).
61  *
62  * Option names are split by pipes('|'), so the
63  * earlier example can be described as the following
64  * string from which the OptionString is constructed:
65  * `"quiet|silent|q|s"`
66  *
67  * @note Empty string such as `"opt||o"` are ignored.
68  */

85  * - The std::string of one acceptable name for the option
