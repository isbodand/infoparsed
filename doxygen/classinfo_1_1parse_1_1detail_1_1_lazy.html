Detailed Description

template<class T, class... TArgs>
class info::parse::detail::Lazy< T, TArgs >

A class that removes the eager instantiation required by RAII to take place.

Instantiates an instance of type T on the heap, whenever the value is first requested. If never requested it will never be instantiated.
